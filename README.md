# 💎 Ayra — Premium Artificial Jewellery Landing Page

> *"Elegance in Every Detail."*

A fully responsive, visually rich **jewellery brand landing page** built with pure **HTML, CSS & Vanilla JavaScript** — no frameworks, no dependencies.

---

## 🌟 Live Preview


> 🔗 https://ayra-pi.vercel.app/

---

## 📸 Overview

**Ayra** is a fictional premium artificial jewellery brand. This project is a front-end landing page that showcases the brand's collections, products, story, and more — designed with a luxury aesthetic, smooth animations, and a fully mobile-responsive layout.

---

## ✨ Features

- 🎬 **Animated Preloader** — Brand logo fade-in before page loads
- 🧭 **Sticky Navbar** — Scrolls with glassmorphism effect + active section highlighting
- 🍔 **Mobile Hamburger Menu** — Smooth drawer with body scroll lock
- 🦋 **Hero Section** — Parallax mouse-move effect on background decorative shapes
- 🏷️ **Features Bar** — Free Shipping, Trendy Designs, Gift Packaging, Skin Friendly
- 📖 **About / Our Story** — Brand narrative with image composition and feature checklist
- 🛍️ **Collections Section** — Earrings, Rings, and Necklaces with hover overlays and badges
- 🗂️ **Category Browser** — Interactive category grid with icon images
- 🔥 **Best Sellers** — Product grid with **live category filter** (All / Rings / Necklaces / Earrings / Bracelets)
- ⏳ **Offer Banner + Live Countdown Timer** — Persists across sessions via `localStorage`
- ✅ **Why Choose Us** — Six-card grid highlighting brand USPs
- 💬 **Testimonials** — Customer review cards with star ratings
- 🔢 **Counter Animation** — Animated number roll-up on scroll
- 🛠️ **Process / How It Works** — Alternating timeline (Design → Deliver)
- 📸 **Instagram Gallery** — Masonry-style image grid with Instagram overlay
- 📬 **Newsletter Form** — Email subscribe with confirmation alert
- 🎞️ **Brand Marquee** — Infinite auto-scrolling trusted-by logos
- 🔝 **Back to Top Button** — Smooth scroll back to top
- 🌄 **Smooth Page Reveal** — Fade-in on page load

---

## 🛠️ Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Structure  | HTML5 (Semantic)                    |
| Styling    | Vanilla CSS3 (Custom Properties, Flexbox, Grid, Animations) |
| Logic      | Vanilla JavaScript (ES6+)           |
| Fonts      | Google Fonts — Playfair Display, Poppins, Cormorant Garamond |
| Icons      | Font Awesome 6                      |
| Storage    | localStorage (for countdown timer)  |

> ⚡ Zero frameworks. Zero build tools. Zero dependencies.

## 🎨 Design Highlights

- **Color Palette** — Warm gold tones with deep charcoal backgrounds
- **Typography** — Playfair Display for headings (editorial luxury feel), Poppins for body text
- **Micro-animations** — Scroll-triggered fade/slide via custom `data-aos` attributes
- **Hover Effects** — Product card actions, collection overlays, gallery Instagram overlays
- **Glassmorphism** — Navbar blur-background on scroll
